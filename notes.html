The World Wide Web

HMTL stand for HyperText Markup Language and is how browsers read web pages.
In addition to your browser and computer, there is the internet and also servers.
All of this web communication happens through HTTP-HyperText Transfer Protocol.

I learned several HTML elements in this lesson. Most HTML elements have opening and closing tags, except for void tags, which don't require closing tags. I also learned how to make links to websites and link pictures
Thankfully, text will be displayed, if the picture fails to load.
The second-to-last thing I learned was the difference between inline and block elements. Block elements form an invisible box around text, like the p tags wrapping this text. The br tag I used four times above, is an inline tag.

Block elements: p, div, form

Inline elements: a, span, br, image, strong

The last thing i learned was document structure. I hope this document in structured correctly. You must declare your doctype, and wrap your document in opening and closing html tags.
It is also important to have a title for the browser and reader.

It helps to think of HTML as organized in a tree format. Within each tag, there may be more tags, like an image or text. The styling is done by CSS, and everything is put in boxes. It is easy to think of a webpage as a number of boxes. Often, there are boxes inside boxes.
